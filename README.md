# 🚀 Getting Started

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/aiforge.git
```

---

## 2️⃣ Install Frontend

```bash
cd client
npm install
```

---

## 3️⃣ Install Backend

```bash
cd server
npm install
```

---

## 4️⃣ Setup Environment Variables

### Backend `.env`

```env
DATABASE_URL=
JWT_SECRET=
GROQ_API_KEY=
```

---

## 5️⃣ Run Backend

```bash
npm run dev
```

---

## 6️⃣ Run Frontend

```bash
npm run dev
```
